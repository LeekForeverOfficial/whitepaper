## 執行摘要

LeekForever 是一個基於區塊鏈技術的創新型社會實驗與去中心化經濟生態系統。項目通過重新定義"韭菜"文化符號，構建了一個融合永久鎖倉機制、靈魂綁定身份與動態釋放經濟的獨特模型。核心創新在於將傳統加密貨幣項目的零和博弈轉變為所有參與者的正和遊戲，特別是為忠實社區成員鋪設了清晰的共同致富路徑。

構建一個自主進化、利益共享、價值永續的去中心化數位國度，其中：

- 早期貢獻者獲得與其付出相匹配的持續回報

- 價值創造與價值分配實現完美閉環

### 1.3 核心創新亮點

1. **永恆鎖倉池** - 不可撤銷的信任基石
2. **動態釋放經濟** - "薪火相傳"的增長引擎

## 3. 核心機制詳解

### 3.1 "永恆之池"與"薪火相傳"動態釋放模型

#### 3.1.1 創世階段（0-33,333人）

- 前33,333名用戶完成永久鎖倉
- 建立約1,000,000美元的基礎流動性
- 每位參與者獲得創世SBT身份憑證

#### 3.1.2 飛升階段（≥33,334人）

**動態釋放觸發條件**：

- 當鎖倉總人數達到33,333時系統啟動
- 第33,334名及之後每個新鎖倉者觸發釋放機制

**貢獻積分競標系統**：

```
貢獻度 = 邀請力(50%) + 忠誠度(30%) + 活躍度(20%)

其中：
├── 邀請力：直接邀請並成功鎖倉人數
├── 忠誠度：持有时長係數 × 基礎分數
└── 活躍度：治理參與 + 持幣量 + 社區貢獻
```

**釋放流程**：

1. 新用戶N完成鎖倉，注入50U流動性
2. 系統開啟一輪"飛升競價"
3. 現有鎖倉者消耗貢獻分參與競標
4. 系統選擇【貢獻分 × 消耗比例】最高者
5. 選中者獲得50U流動性釋放，額外獎勵，SBT標記"已飛升"
6. 池中鎖倉人數保持33,333不變

### 3.2 靈魂綁定代幣（SBT）賦能體系

#### 3.2.1 SBT作為數位身份

- **唯一性**: 每個地址限鑄1個，不可轉讓
- **紀念性**: 記錄創世貢獻與加入時間
- **功能性**: 所有權益的訪問憑證

#### 3.2.2 三層權益結構

**第一層：現金流權利**

- 定期獲得交易稅分紅（穩定幣）
- IP商業利潤的穩定收益分配
- 生態項目收益的現金分紅

**第二層：資產增值權利**

- 優先獲得生態項目空投（平均3-5%總量）
- 社區金庫投資的利潤分成
- 專屬NFT系列的免費鑄造權

**第三層：治理與特權**

- 加權投票權
- 提案發起與投票權
- IP衍生品優先購買權
- 線下活動專屬資格

### 3.3 "澆水計劃"3.0 - 貢獻證明增長引擎

#### 3.3.1 多維度貢獻評估

```
綜合貢獻分 =
├── 增長貢獻（40%）：邀請新用戶、社區推廣
├── 資本貢獻（25%）：持幣量、流動性提供
├── 治理貢獻（20%）：提案、投票、討論
└── 創意貢獻（15%）：內容、程式碼、藝術創作
```

#### 3.3.2 反女巫攻擊機制

- 行為模式分析算法
- 多因子身份驗證
- 漸進式獎勵釋放

### 3.4 外部投資者參與生態

#### 3.4.1 二級市場參與路徑

- 通過DEX直接購買$LEEK
- 參與流動性挖礦獲得獎勵
- 持有代幣享受通縮增值效應
